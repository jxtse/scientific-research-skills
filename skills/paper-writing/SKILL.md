# Paper Writing

Structured workflow for academic paper writing and revision.

## Status: 🚧 Draft

Core workflow being codified from camera-ready experience (ACL 2026 CAST paper):

1. **Outline** → Structure the story before writing
2. **Draft** → Write section by section, figures first
3. **Internal review** → Self-critique with specific checklist
4. **Revision** → Address weaknesses found in review
5. **Camera-ready** → Final formatting, references, appendix

Key lessons from practice:
- Write the abstract last
- Figures and tables tell the story — make them standalone
- Related work is a conversation, not a list
- Limitations section builds credibility
