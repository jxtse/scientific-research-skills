# Experiment Design

Systematic experiment planning for research papers.

## Status: 🚧 Draft

Core principles being codified:

1. **Baselines must be fair** — same compute budget, same hyperparameter search
2. **Ablations reveal contribution** — remove one component at a time
3. **Metrics must be appropriate** — don't use accuracy when you mean F1
4. **Statistical significance** — report confidence intervals, not just point estimates
5. **Reproducibility** — document every parameter, random seed, hardware spec

More detailed workflow coming from ongoing practice.
